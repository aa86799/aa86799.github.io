typdef与#define的区别

#define 是用来定义宏的。而宏的语义就是在预编译时，用宏定义后的字符串来替换宏名。

如，

typedef char* String;， 可以用宏来替换 #define String char*

但是，像一些复杂的别名定义，如与结构体、枚举等这样的 复合类型的定义，就不能直接使用#define来定义了。当然，若是分段来定义，还是可以的。结合上面枚举的例子，如:
